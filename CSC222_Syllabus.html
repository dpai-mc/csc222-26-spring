CSC222: Programming and Algorithms II

This course focuses on software engineering techniques, data structures (arrays, records, strings, linked lists, stacks, queues, hash tables), recursion, and object-oriented programming concepts. Students will learn to design, implement, test, and debug programs using modern object-oriented languages.

  • Translate basic Java constructs into C++
